# Planning Poker
___
#Infnet #InstitutoInfnet #Java #PlanningPoker #Agile #Scrum #Estimativa

História
-
___
<p>Este projeto surgiu como uma ideia durante uma sessão de planejamento de uma sprint em um dos projetos que participei. Fiz uma busca rápida por serviços 
online de planning poker, e a grande maioria dos que encontrei eram serviços pagos, com opções restritas de forma gratuita. As ferramentas 100% gratuitas 
também possuíam limitações em diversos aspectos, deixando a desejar em facilidades úteis para o planejamento.</p>
<p>O projeto será desenvolvido durante o andamento do curso de pós graduação MIT em Engenharia de Software com Java no Instituto Infnet</p>

Stack utilizada
-
___
<p>Para este projeto será utilizado o <b>Java</b> como linguagem de programação, na sua versão 18, junto com o <b>framework Spring</b> <i>(Spring Boot, Spring Web 
(MVC), Spring Data JPA, Spring Security, Spring Cloud)</i>.</p> 
<p>Para o lado do usuário ainda não temos uma definição de tecnologia a ser utilizada.</p> 
<p>Para a camada de persistência será utilizado o banco de dados relacional <b>Postgresql</b>.</p>
<p>Complementando a stack, temos o <b>Git</b> como ferramenta de versionamento, e o <b>Github</b> como repositório, além do <b>Intellij IDEA</b> como IDE utilizada no projeto.</p>
